Getting the Track Play History¶

The play history can be retrieved via /history. Pagination is supported through the arguments ipp, which specifies the number of items per page of results, and page, which specifies which page of results to get.

partify.history.history()[source]

An endpoint that provides playback history.

Parameters:
  • ipp (string or integer) – Specifies the number of history items per page. Defaults to 25.
  • page (string or integer) – Specifies the page of history items to get. Defaults to 1.
Returns:

A JSON structure representing the track history for the period specified by the client.
Return type:

A JSON string

Route: /history

Methods: GET

Example

GET /history?ipp=10&page=1 HTTP/1.1
