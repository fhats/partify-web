Getting Statistics¶

A set of statistics about the player history can be retrieved from /statistics . Statistics for the last day, week, month, year, and for all time will be returned.

partify.statistics.statistics()[source]

Provides some statistics about the partify.models.PlayHistoryEntry in the database.

Returns:The status of the request, the time the statistics were generated, and the statistics. The statistics are divided up into segments by time.
Return type:JSON string

Route: /statistics

Methods: GET
