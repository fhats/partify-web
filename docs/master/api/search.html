Searching the Library¶

To search through the Mopidy catalog for available tracks, send a GET request to /track/search with at least one of the following parameters:

  • artist - Specifies the artist to search for
  • title - Specifies the title of the track to search for
  • album - Specifies the album to search for

A list of results will be provided, from which you can find Spotify URLs to use with /queue/add . Results will be returned as sorted by a crude relevance function, then by artist, then by album.

partify.track.track_search(*args, **kwargs)[source]

Performs a search on the MPD server for tracks matching the specified criteria (as arguments in the URL i.e. HTTP GETs). Must contain at least one of artist, title, or album.

Parameters:
  • artist (string, optional) – The artist to search for
  • title (string, optional) – The title to search for
  • album (string, optional) – The album to search for
Returns:

A list of search results
Return type:

JSON string

Route: /track/search

Methods: GET

Examples

GET /track/search?artist=Lemuria&album=Get+Better HTTP/1.1
