Statistics¶

An endpoint and helper functions for computing statistics from the play history in the database.

partify.statistics.compute_stats_over_segment(segment)[source]¶

Computes some statistics over a segment of history. Note that this could be pretty slow to start. May be worth finding a way to speed this up eventually.

Parameters:segment (list of partify.models.PlayHistoryEntry) – A list of partify.models.PlayHistoryEntry to compute statistics over.
Returns:A dictionary of statistics for this segment.
Return type:dictionary
partify.statistics.statistics()[source]¶

Provides some statistics about the partify.models.PlayHistoryEntry in the database.

Returns:The status of the request, the time the statistics were generated, and the statistics. The statistics are divided up into segments by time.
Return type:JSON string

Route: /statistics

Methods: GET
