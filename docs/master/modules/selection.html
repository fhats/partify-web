Selection¶

Contains the selection schemes that are used to negotiate the ordering of the global queue.

partify.selection.first_come_first_served(mpd, db_tracks)[source]¶

Simple first-come first-served ordering. Generates a list of users based on when they queued tracks and then lets _match_tracks_with_users() do the legwork.

Parameters:
  • mpd (MPDClient) – An MPD client to use to manipulate the MPD list
  • db_tracks (list of partify.models.PlayQueueEntry) – A list of tracks in the database to be compared against
partify.selection.first_come_first_served_with_voting(mpd, db_tracks)[source]¶

First-come first-served with voting. Sorts the partify.models.PlayQueueEntry in the database and shuffles tracks until their order matches the correct DB ordering.

Parameters:
  • mpd (MPDClient) – An MPD client to use to manipulate the MPD list
  • db_tracks (list of partify.models.PlayQueueEntry) – A list of tracks in the database to be compared against

partify.selection.get_users_next_pqe_entry_after_playback_priority(user_id, playback_priority)[source]¶

Gets the user’s next PlayQueueEntry that has a playback priority higher than the given priority. Useful for determining if a user’s tracks are out-of-order w.r.t. the user’s queue priorities.

Parameters:
  • user_id (User) – The id of the User to examine
  • playback_priority (integer) – The priority after which to look for the user’s next track
Returns:

The next PlayQueueEntry in the user’s queue
Return type:

PlayQueueEntry
partify.selection.round_robin(mpd, db_tracks)[source]¶

Organizes a round-robin arrangement of tracks in the global play queue. Generates a list of users in round-robin order starting with the user playing the current track and proceeding in alphabetical order by username. Sends the user list and db_tracks off to _match_tracks_with_users() to be reordered.

Parameters:
  • mpd (MPDClient) – An MPD client to use to manipulate the MPD list
  • db_tracks (list of partify.models.PlayQueueEntry) – A list of tracks in the database to be compared against
