Decorators¶

A collection of decorators for common endpoint actions.

partify.decorators.with_authentication(f)[source]¶

A decorator that ensures that a user is logged in and redirects them to a login page if the user is not authenticated.

The way authentication is currently verified is simply by checking for the user key in session. This is certainly not the best means of authentication but it will do for now. Security is not a high priority for this project since its intended use case is in a local scenario. However, it would be great to have better security if time permits.

partify.decorators.with_mpd(f)[source]¶

A decorator that establishes an MPD connection Mopidy and passes it into the wrapped function.

partify.decorators.with_mpd_lock(f)[source]¶

A decorator that ensures that no two functions that send commands to MPD happen simultaneously.

partify.decorators.with_privileges(privs, fail_mode='json')[source]¶

A decorator that restricts access to the decorated endpoint if the logged in user does not have sufficient privileges.
